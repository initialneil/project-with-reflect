# project-with-reflect

Use `skills/project-with-reflect/SKILL.md` when the user asks to register, check in to, manage, or
reflect on projects, machines, devices, APIs, MCP servers, or reusable knowledge.

Before acting on a managed project or connection, load its generated `SKILL.md`, dashboard note,
decisions, and relevant lessons. Log notable work, record durable results immediately, and close
with reflect so future sessions start from distilled lessons instead of raw chat history.
