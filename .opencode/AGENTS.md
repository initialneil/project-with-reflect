# project-with-reflect

Use the canonical instructions in `skills/project-with-reflect/SKILL.md`.

This skill keeps personal project memory in `$PROJECT_WITH_REFLECT_ROOT`: generated project and
connection skills, logs, lessons, decisions, and Obsidian-friendly dashboards.

When working with a registered project or connection, load first, check prior decisions, operate
connections through their own skills, log notable events, record durable findings, and run reflect
at session end.
