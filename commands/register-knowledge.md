---
description: "Register a global, agent-usable knowledge module (project-with-reflect)"
---
Run the **register-knowledge** action of the `project-with-reflect` skill with arguments: $ARGUMENTS

Creates a global module under `$PROJECT_WITH_REFLECT_ROOT/knowledge/<name>/` (e.g. an MCP/API
setup) that any project opts into via `config.json.knowledge`. Follow the skill's steps.
